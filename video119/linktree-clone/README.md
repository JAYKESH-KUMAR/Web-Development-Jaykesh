# 🎬 Netflix Clone (Frontend Project)

A responsive Netflix homepage UI clone built using **HTML, CSS, and JavaScript**.  
This project replicates the visual layout and design of Netflix India for learning and portfolio showcase purposes.

---

## 🚀 Live Demo
🔗 https://jaykesh-kumar.github.io/Web-Development-Jaykesh/video45-netflix-clone/

---

## 🛠️ Tech Stack
- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- JavaScript (Basic DOM usage)  
- Git & GitHub  
- GitHub Pages (for deployment)

---

## ✨ Features
- Responsive Netflix-like UI  
- Hero section with call-to-action  
- Video background sections  
- FAQ section with hover effects  
- Mobile-friendly layout
